# MigrAves Climate Sentinel

## Capa biológica para el estudio climático

MigrAves convierte observaciones de aves migratorias, variables ambientales y líneas base locales en registros trazables que pueden acumularse y revisarse para futuros estudios.

### Problema

El clima se mide con instrumentos físicos, pero sus efectos también se manifiestan en los ciclos de los seres vivos. Hoy muchas observaciones de aves quedan fragmentadas en imágenes, sensores y planillas incompatibles. Una observación aislada no prueba cambio climático; una serie estandarizada y validada sí puede transformarse en material útil para investigación.

### Tres productos

1. **ABM - Área Bioclimática Migratoria:** territorio definido por ecosistema, coordenadas, radio, especies prioritarias, línea base, responsables y acceso.
2. **RBM - Registro Bioclimático Migratorio:** evento estructurado con observación biológica, microclima, procedencia, incertidumbre y validación.
3. **IRBC - Índice de Respuesta Biológica al Clima:** indicador experimental de 0 a 100 con componentes de fenología, abundancia, hábitat y meteorología.

### Flujo

Crear ABM -> observar -> comparar -> generar RBM e IRBC -> revisar -> exportar -> acumular serie temporal.

### Rol humano

La IA propone y estructura. Estudiantes y técnicos revisan. Biólogos y científicos validan los registros relevantes. Las correcciones permanecen vinculadas al dato.

### Calidad

- N2: estimación de IA pendiente.
- N3: revisión humana.
- N4: validación profesional explícita.
- N0: registro descartado.

### Formatos

CSV para análisis estadístico, JSON para integración y GeoJSON para sistemas geográficos. Las coordenadas sensibles se generalizan.

### Piloto

Tres áreas iniciales en el sur de Chile: Chamiza, Maullín y Chiloé. La fase física futura incorpora cámaras calibradas, bioacústica, sensores meteorológicos y de hábitat, procesamiento local, energía solar y almacenamiento geoespacial.

### Impacto

- datos estructurados para estudios futuros;
- formación práctica para estudiantes;
- roles para biólogos, técnicos y curadores de datos;
- alertas para priorizar seguimiento;
- conciencia comunitaria sin exponer biodiversidad sensible.

### Límite científico

MigrAves no afirma que las aves predigan tornados o desastres, ni que una observación pruebe cambio climático. El proyecto crea infraestructura de datos para estudiar tendencias mediante muestreo repetido, calibración, validación y análisis estadístico.

> Las estaciones miden la atmósfera. MigrAves registra cómo la vida está respondiendo.
