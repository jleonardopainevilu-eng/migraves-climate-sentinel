# Submission checklist

## Product

- [ ] Open the deployed URL on Android and desktop.
- [ ] Create a custom ABM.
- [ ] Run the no-key Chamiza simulation.
- [ ] Confirm that an RBM and IRBC are generated.
- [ ] Test N3 review and the N4 role guard.
- [ ] Export CSV, JSON, and GeoJSON.
- [ ] Verify coordinate generalization on a restricted/sensitive area.
- [ ] Test one real image with the configured OpenAI API.
- [ ] Confirm that errors do not expose the API key.

## Repository

- [ ] Public GitHub repository.
- [ ] Files are in the repository root, not inside a second nested folder.
- [ ] README explains setup, scientific limits, and architecture.
- [ ] `npm run check` passes.
- [ ] `npm test` passes.
- [ ] No `.env.local` or API key is committed.
- [ ] Add screenshots and the final deployed URL to README.
- [ ] Add the main Codex session identifier required by the competition form.

## Video

- [ ] Show the problem in less than 25 seconds.
- [ ] Create/select an ABM.
- [ ] Run the full RBM + IRBC demonstration.
- [ ] Show provenance, uncertainty, and limitations.
- [ ] Show human validation and data-quality levels.
- [ ] Export at least one research format.
- [ ] Explain exactly where GPT-5.6 and Codex are used.
- [ ] Clearly separate current MVP from future physical hardware.
- [ ] Keep the video within the official duration requirement.

## Scientific communication

- [ ] Never state that one record proves climate change.
- [ ] Never claim tornado, earthquake, or disaster prediction.
- [ ] Describe the IRBC as experimental.
- [ ] Describe taxonomy and counts as estimates until reviewed.
- [ ] Explain that production N4 requires authenticated professional credentials.
- [ ] Protect sensitive biodiversity coordinates.

## Final form

- [ ] Project title: MigrAves Climate Sentinel.
- [ ] One-sentence proposition is consistent across form, video, site, and README.
- [ ] Repository URL works without authentication.
- [ ] Deployment URL works without authentication.
- [ ] Video URL is public or unlisted as allowed by the rules.
- [ ] All required fields and acknowledgements are complete.
- [ ] Re-check the official competition page for deadline and rule updates before submitting.
