# Changelog

## 0.3.0 - Climate data-product MVP

- Renamed the product to **MigrAves Climate Sentinel**.
- Added the **ABM - Area Bioclimática Migratoria** builder.
- Added custom-area persistence in browser storage.
- Added ABM JSON export and access classification.
- Reframed output as a standardized **RBM - Registro Bioclimático Migratorio**.
- Added experimental **IRBC - Índice de Respuesta Biológica al Clima** with phenology, abundance, habitat, and weather components.
- Added N0-N4 data-quality workflow with named reviewer and role.
- Restricted N4 UI validation to biologist/scientist roles.
- Added CSV, JSON, and GeoJSON exports.
- Added coordinate generalization for restricted and sensitive areas.
- Added a local 30-record registry.
- Added formal ABM and RBM JSON Schemas and an example record.
- Updated the OpenAI structured-output schema.
- Updated competition description, video script, architecture, and scientific guardrails.

## 0.2.0 - Sentinel network MVP

- Added three pilot nodes, provenance labels, historical comparison, validation, and export.
